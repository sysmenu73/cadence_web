Allegro PCB Designer

快速实现设计从概念到投产

 

 

Key Benefits

  • 加速从布局布线到生产制造的设计过程
  • 提供一系列强大功能，例如设计分区、RF 设计功能和互连设计规划
  • 经验证的生产能力，可提高生产力，帮助工程师快速实现批量生产
  • 通过 TÜV SÜD “Fit for Purpose – TCL1”认证，符合 ISO 26262 汽车功能安全要求

 

 

Cadence® Allegro® PCB Designer 采用约束驱动型设计系统，快速将无论简单或是复杂的概念设计转化为产品，同时确保设计的功能性和可制造性。使用扩展技术工具包，设计人员能够高效、经济地满足当前和将来的所有技术和方法需求，并且适用于各种规模和复杂性的团队、组织和项目。

 

Allegro PCB Designer 拥有业界领先的高速设计、RF 天线、柔性板和可制造性设计 (DFM) 技术，集各种功能于一身，帮助您交付使客户满意的创新设计和产品。高效先进的复用设计和可生产功能模块，是Allegro PCB Designer 提供的独特优势，让您在竞争中领先对手，在市场中独占鳌头。

 

全设计流程的强大功能（例如设计分区、RF 设计功能和互连设计规划），可以让设计从布局、布线到生产制造的效率大大提升

 

经过 TCL1 认证，符合 ISO 26262 汽车安全要求

业内首个获得“Fit for Purpose – TCL1”认证的 PCB 设计和验证流程，满足严格的 ISO 26262 汽车安全要求。该流程使用 PSpice®、Allegro 和 OrCAD®产品套件，涵盖从设计到仿真，以及物理实现和验证在内的全套功能。高性能的设计输入、仿真和版图设计工具为设计工程师提供了一个集成环境，使您可以针对各个电路规范进行安全规范验证，提高设计可靠性。相关安全手册、Tool Confidence Analysis (TCA) 文档以及 TÜV SÜD 合规性报告的信息，请通过 Cadence 在线支持页面下载功能安全文档包。

 

主要功能

  • 提供功能丰富、可灵活扩展的 PCB 完整设计解决方案
  • 约束驱动的设计流程，减少设计迭代
  • 集成 DesignTrue DFM 技术，提供实时 DFM 检查
  • 提供从前端到后端的单一约束管理环境，确保设计一致性
  • 支持先进的刚柔结合设计，最大程度减少柔性和刚柔结合设计的设计迭代，降低设计成本
  • 增强动态同步团队协同设计能力，大大减少花费在布线、调整和优化信号上的时间，从而缩短设计周期
  • 支持集成的 RF/模拟设计和混合信号设计环境
  • 提供交互式布局规划和组件布局
  • 支持大规模、人员分散的开发团队所需的设计分区
  • 支持实时、交互式推/挤走线编辑功能
  • 支持动态铜皮实时编辑优化
  • 约束管理器支持网络拓扑设定、时序、串扰、走线层和其他走线，间距约束设定和检查
  • 集成可靠的 PCB 布线器，可对随机信号进行自动布线
  • 支持分层布线规划，可加速完成设计
  • 配置高速接口，可缩短高密度设计的互连规划和布线时间

  • 提供全面、强大、易于使用的工具套件，有效且成功地帮助设计师实现从设计到制造的过渡：

    • DFM Checker 可基于公司/生产制造合作伙伴所提供的约束进行验证
    • Documentation Editor 可减少多达 70％ 的生产制造文档所花费的时间。
    • Panel Editor 可将拼板设计意图传达给生产制造合作伙伴
  • 支持各种制造格式输出设计数据
